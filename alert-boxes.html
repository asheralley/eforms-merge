ALERT BOXES - ALERT WARNING / TOASTER API

Create

- To create a toaster use uiMods.toaster(event, {options})

- An event object must be passed as the first parameter every time the function is invoked: uiMods.toaster(event).This can simply be the word event representing the event object

Options

- The toaster has preset defaults, so the 2nd options object/parameter {options} is optional. If an options object is not included in the function call, the toaster will function with all defaults.

- List of options uiMods.Toaster(event, {options}) and their defaults:

  • text: 'string' - default = 'This is a warning'
  • timer: boolean - default = false
  • speed: integer - default = 3500
  • color: 'string' - 'light'/'dark' - default = 'light'

Invoking

- As an example the third button above: INLINE FUNCTION CALL ALERT, uses the following inline invoke pattern:

  • uiMods.toaster(event, {text:'I can be easily changed and adjusted', timer: true, speed: 5000, color: 'dark'})

- Or via js notice 'event' object being passed twice when the addEventListener method is used. This pattern is needed to pass arguments to a function via the eventListener() method.

  • addEventListener('click', function(event){uiMods.toaster(event, {text: 'This is an alert with a TIMER', timer: true, speed: 3000, color: 'dark'})}, false);
